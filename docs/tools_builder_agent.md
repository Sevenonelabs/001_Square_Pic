# SquarePic.io Tools Expansion Agent Guide

You are responsible for expanding `SquarePic.io` from a single square image tool into a clean, high-converting creator utility suite.

Your job is to seamlessly add:

1. Watermark Tool
2. Background Remover Tool

while preserving the premium, minimal, fast, browser-based experience of the current product.

This is NOT Canva.
This is NOT Photoshop.
This is NOT a bloated dashboard full of nonsense.

This must remain:

- fast
- simple
- privacy-first
- browser-based
- no signup required
- creator-focused
- SEO optimized
- premium looking
- mobile responsive
- conversion focused

Do not overengineer.
Humans already do enough of that.

---

# Core Product Philosophy

Every tool must answer:

## "Can the user finish this in under 30 seconds?"

If no:
you are building the wrong thing.

The product promise is:

## Instant creator utilities for social media and business visuals.

Not:

## complex design software

Avoid feature bloat.

---

# Existing Product

Current live tool:

## Square Image Tool

Functions:

- convert image to square format
- blur background
- solid background
- crop to fill
- platform-friendly export

This remains the hero product.

New tools must feel like natural extensions.

# New Tool 1: Watermark Tool

## Required Features

### Add Text Watermark

Allow users to:

- add custom text
- choose font size
- choose font color
- opacity control
- position control
- rotate watermark
- repeat watermark pattern (optional if simple)

Keep UI minimal.

Do not offer 500 fonts because apparently humans believe typography is a personality trait.

Use 5–8 premium font options maximum.

---

### Add Logo Watermark

Allow users to:

- upload PNG logo
- resize logo
- opacity control
- placement control
- margin control

Support transparent PNG uploads.

Preview must be real-time.

---

### Remove Watermark (Careful)

Only allow:

## simple object cleanup for user-owned content

Do NOT market this as piracy support.

Avoid language like:

- remove copyrighted watermark
- remove protected branding

Use safer phrasing:

## clean unwanted marks from your own images

This is important.

We are building a tool, not assisting internet crime.

---

## Export Options

Support:

- JPG
- PNG
- HD export

Keep export fast.

Primary CTA:

## Download Watermarked Image

---

# New Tool 2: Background Remover Tool

## Required Features

### Remove Background

Allow users to:

- upload image
- auto-remove background
- preview cutout instantly

Must support:

- portraits
- products
- profile photos
- creator thumbnails

Fast result matters more than fancy UI.

---

### Transparent PNG Export

Critical.

This is the main value.

Primary export:

## Download Transparent PNG

---

### Solid Background Replacement

Allow users to:

- add white background
- black background
- brand color background
- custom HEX color
- simple gradient presets (optional)

Do not turn gradients into a design festival.

Keep it clean.

# Design Rules

All tools must match the current SquarePic design system:

- dark premium UI
- soft glow accents
- clean spacing
- strong visual hierarchy
- premium typography
- polished hover states
- glassmorphism where useful
- subtle depth
- smooth transitions

Inspired by:

- Linear
- Framer
- Raycast
- Notion
- Figma

Avoid:

- cheap dashboard feeling
- generic admin panel styling
- template-looking sliders
- cluttered sidebars

Users should feel:

## "this is premium"

not

## "this was made by a tutorial video"

# Layout Rules
