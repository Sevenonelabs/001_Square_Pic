✦ Professional Troubleshooting

Image Upload & Processing

  • Ensure you are using modern formats: WebP, JPEG, PNG, GIF, or AVIF.
  • For images over 20MB, we recommend light compression before upload for the best browser performance.
  • If the "Select Image" button doesn't respond, try dragging and dropping your file directly onto the editor.
  • If an error persists, a simple browser refresh often resolves local script conflicts.

Export & Quality Settings

  • If your final export looks different from the preview, refresh the page and re-upload, as this synchronizes the browser's rendering engine.
  • The quality slider is specifically for lossy formats (JPEG/WebP); it is expected to be disabled for lossless PNG exports.

Interface & Display

  • Our editor is fully responsive. If some controls are not visible, use the scrollable sidebar or adjust your browser zoom (Ctrl/Cmd + −).
  • Ensure your browser has permission to save files if the download doesn't start automatically.

Technical Loading

  • For any loading errors, clear your browser cache and perform a hard-refresh (Ctrl/Cmd + Shift + R).
  • Disable extensions that may block scripts or cross-origin requests.
