✦ Square Image Download Issues

If your final image is not downloading after you click export, please try the following steps:

  • Check your format: Try exporting as a JPEG instead of PNG.
  • Refresh the page: Sometimes reloading the editor clears memory issues.
  • Check permissions: Ensure your browser has permission to download files to your device.

✦ Browser Support & Mobile Compatibility

Our online square photo editor works directly in your web browser. There is no app to download.

  • Desktop: Works best on modern versions of Chrome, Safari, Firefox, and Edge.
  • Mobile compatibility: Fully supported on iOS (Safari/Chrome) and Android (Chrome) to easily create square images for Instagram.

If the tool feels slow or unresponsive, make sure you are not using an outdated browser version.

✦ Supported File Types

✦ Privacy Reassurance
