✦ Frequently Asked Questions

What size is a Twitch banner?

The recommended size for a Twitch profile banner is 1200 x 480 pixels. This is a very wide aspect ratio. Video offline banners require a 16:9 ratio (1920 x 1080).

How do I make my art fit the banner?

Because the 1200 x 480 aspect ratio is so wide, normal photos will be chopped off. You can use our tool to extend the background of your artwork horizontally so it fits the massive banner space without ruining the core image.

Is there a file size limit?

Yes, Twitch restricts profile banners to 10MB. If your artwork is too heavy, use the compression slider in SquarePic to reduce the weight before saving.
